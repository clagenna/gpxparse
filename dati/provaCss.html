Set frozen set

>>> nums = {10, 20, 30, 40}  # nuovo set di 4 elementi
>>> nums  # gli elementi del set non sono ordinati
{40, 10, 20, 30}

I set vengono definiti elencando tra parentesi graffe ({}) una serie di oggetti separati da virgole (,). Nonostante sia set che dizionari usino le parentesi graffe, Python è in grado di distinguerli perché i dizionari contengono chiavi e valori separati dai due punti.

Dato che la sintassi {} è già utilizzata per definire un dizionario vuoto, non c’è nessuna sintassi disponibile per creare un set vuoto, che si può però definire usando set(). Anche per il frozenset non esiste una sintassi dedicata, ma è possibile usare frozenset(...).

>>> setAnd = mySet & mySet2
>>> setAnd

OR

>>> setOr = mySet | mySet2
>>> setOr

Differenza

OR esclusivo

>>> mySet ^ mySet2

for e for-else

Il ciclo for ci permette di iterare su tutti gli elementi di un iterabile ed eseguire un determinato blocco di codice. Un iterabile è un qualsiasi oggetto in grado di restituire tutti gli elementi uno dopo l’altro, come ad esempio liste, tuple, set, dizionari

seq = [1, 2, 3, 4, 5]
for n in seq:
    print('Il numero', n, 'è', end=' ')
    if n%2 == 0:
        print('pari')
    else:
        print('dispari')

le comprehention

ci permette di creare in modo conciso e conveniente nuove liste, set, e dizionari partendo da una sequenza di valori esistenti.

list comprehention

>>> # list comprehension che crea una lista di quadrati
>>> [x**2 for x in range(10)]
[0, 1, 4, 9, 16, 25, 36, 49, 64, 81]

l'esercizio del for con la comprehention

>>> lip2 = [ n for n in range(10,30) if n%2 == 0 and n<25 ]
>>> lip2
[10, 12, 14, 16, 18, 20, 22, 24]

oppure con filtro: listcomp che filtra i numeri e prende solo quelli pari

>>> [x for x in range(10) if x%2 == 0]
[0, 2, 4, 6, 8]

listcomp che crea tutte le combinazioni tra ABC e 123

>>> [c+n for c in 'ABC' for n in '123']
['A1', 'A2', 'A3', 'B1', 'B2', 'B3', 'C1', 'C2', 'C3']

set comprehention

>>> # set comprehension che crea un set di cubi
>>> {x**3 for x in range(10)}
{0, 1, 64, 512, 8, 343, 216, 729, 27, 125}

altro esempio di set comprehention

>> a='doppione'
>>> b={x for x in a}
>>> b
{'d', 'o', 'p', 'e', 'n', 'i'}

altro esempio, setcomp che filtra le lettere e prende solo quelle uppercase

>>> {c for c in 'aAbBcCdDeE' if c.isupper()}
{'C', 'E', 'B', 'A', 'D'}
>>>

dict comprehention

che ha la seguente sintassi
{key_expr : val_expr for item in iterable if cond }

>>> # dict comprehension che mappa lettere lowercase all'equivalente uppercase
>>> {c: c.upper() for c in 'abcde'}
{'c': 'C', 'e': 'E', 'a': 'A', 'b': 'B', 'd': 'D'}

altro esempio di dict comprehention

>>> a = 'Python'
>>> di1={k: ord(k) for k in a}
>>> di1
{'P': 80, 'y': 121, 't': 116, 'h': 104, 'o': 111, 'n': 110}

Parametri

parametri posizionali

def myFunc( a, b, c, d):
    print(a,b,c,d)

richiamo

myFunc(10,20,30,40)
10 20 30 40

sbaglio quantita argomenti

>>> myFunc(10,20,30)
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
TypeError: myFunc() missing 1 required positional argument: 'd'

richiamo con nomina(keyword)

>>> myFunc(b=10,a=20,d=30,c=40)
20 10 40 30

Parametri opzionali

def myFunc( a, b, c=50, d=80):
    print(a,b,c,d)
>>> myFunc(10,20)
10 20 50 80

Parametri *args

in questo caso args rappresenta una tupla

>>> def myFunc( *args):
...     print(args)
...
>>> myFunc(10,20,30)
(10, 20, 30)

Parametri **kwargs

in questo caso args rappresenta un dict

>>> def myFunc(**kwargs):
...     print(kwargs)
...
>>> myFunc(b=10,a=20,d=30,c=40)
{'b': 10, 'a': 20, 'd': 30, 'c': 40}

...
>>> myFunc(3,5)

Funzioni nidificate

in python le funzioni possono essere nidificate

>>> def outer(x,y):
...     def inner(a,b):
...             return a + b
...     print(inner(x,y))
...
>>> outer(10,5)
15

si possono tornare funzioni

>>> def outer(x,y):
...     def inner(a,b):
...             return a + b
...     return inner
...
>>> fu = outer(2,3)
>>> fu(7,5)
12
